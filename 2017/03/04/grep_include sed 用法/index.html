grep include 使用

作用：缩小搜索范围, 排除/包括文件或者目录

  • –include *{.c,.cpp} 只在目录中搜索.c和.cpp文件；
  • –exclude “README” 排除所有README文件
  • –include-dir 仅在某些目录中搜索
  • –exclude-dir 排除某些目录
  • –exclude-from FILE 从文件FILE中读取需要排除的文件列表

例子

set

  • set -x 是开启

  • set +x是关闭

  • set -o是查看 (xtrace)

  • set 去追中一段代码的显示情况

批量替换

sed -i “s/查找字段/替换字段/g” grep 查找字段 -rl 路径

sed -i “s/查找字段/替换字段/g” ./*

若字符串中包含“/“等特殊字符串，可以:

sed -i “s#查找字段#替换字段#g” grep 查找字段 -rl 路径

sed -i “s#查找字段#替换字段#g” ./*
